# 🎵 Mini Music Player

A simple **music player** built using **HTML, CSS, and JavaScript**.  
It includes basic controls like **Play, Pause, Next, Shuffle, and Volume adjustment**.

## 🚀 Features
- ▶️ Play / ⏸ Pause songs  
- ⏭ Next song  
- 🔀 Shuffle songs  
- 🔊 Adjust volume  

## 🛠️ Tech Stack
- **HTML**  
- **CSS**  
- **JavaScript**
